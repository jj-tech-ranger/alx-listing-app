# ALX Listing App

This project scaffolds a modern Airbnb clone listing page using Next.js, TypeScript, TailwindCSS, and ESLint.

## Project Structure

- **components/common/**: Reusable UI components (Card, Button).
- **interfaces/**: TypeScript interfaces for consistent types.
- **constants/**: Project-wide constants for configuration and labels.
- **public/assets/**: Images and SVGs used across the project.
- **pages/**: Application pages, starting with listing page.
- **styles/globals.css**: Main global styles using TailwindCSS.

## Getting Started

1. Install dependencies:
    ```
    npm install
    ```

2. Run the development server:
    ```
    npm run dev
    ```

3. Open [http://localhost:3000](http://localhost:3000) to view the app.

## Goals

- Clean, scalable codebase for professional production.
- Reusable components and strict type safety.
- Asset and constants management for best practice development.
